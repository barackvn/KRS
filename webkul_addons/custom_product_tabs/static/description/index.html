Features

Using this module admin can classify the product related information in tabs on product view, Tabs based information are very useful for large page content, it helps in making the information more lucrative and classified.

Admin can create many tabs as per product property
like Item Details, Payment Options, Shipping Policy, Product Specifications, etc.

  • Add product infomation from Backend.
  • Describes your product using text, images or tables.
  • Vertical and Horizontal tabs options.
  • Full customizable content.
  • You can modify the order of the product tabs.

Add Product Tabs

After the Installation of this module, you can see the Tabs option on product view, here you can add tabs. You can also decide the tabs view by selecting the tabs type Vertical Tabs or Horizontal Tabs.

Add Product Tabs

You can create the tabs by placing the tab name and tab content on HTML editor, you can place an image, link, table, text.
