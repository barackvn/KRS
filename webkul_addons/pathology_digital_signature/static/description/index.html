Pathology Digital Signature

Patholgy Digital Signature is module by user can upload the signature in Technician and Pathologist profile that will be automatically printed in the Diagnosis Reports of Patient.

Basic Features

  • Digital Signature module provides facility to print signature in Patient Diagnosis Report.
  • Admin can upload the signature of Pathologist and Technician in their record.
  • Signature will be automatically printed in the Diagnosis report if available.
  • Also, Job Position will be shown in report along with the name and signature by this module.
