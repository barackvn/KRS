Customer Credit Limit

Set spending limit for your customers to better manage expenditure for your customers with Odoo customer credit limit module. The admin can set a credit limit for every customer. When the total expenditure for the customer including for all the previous unpaid orders exceeds the credit limit, a warning message pops up to indicate the exceeded limit. Only the manager can approve orders which exceed the credit limit. The Odoo user can send the approval request via mail to the Manager.

Features
  • Set credit limit for a specific customer.
  • Show the detailed info of the customer's outstanding balance.
  • Show all the previous unpaid orders & invoices with the credit limit info.
  • Displays all the non-invoiced orders with the credit limit info.
  • Prompts while approving any order which exceeds the allocated credit limit.
  • A warning message pops up each time when the user tries to confirm an order exceeding the credit limit.
  • A warning message pops up if any previous invoice is pending or due date is passed in customer's account.

Approval request mail.

If a user is not manager then the user can choose the exceed limit options which will send a mail to the manager of the related employee configured in the HR Employee portal.

Approval confirmation mail.

When any manager will approve the order then the module will send a mail to the salesperson of the order.

These emails will also log to the order chatter as a note for future reference.
