Odoo Marketplace Advertisement Manager

Odoo Marketplace Advertisement Manager is a module by which Sellers of Marketplace can promote and advertise their products by purchasing the ad block positions given in the website according to their availability of date.

User Guide

Highlighted Features

  • Admin provides the panels located at different positions on the website to sellers, where they can place their advertisement products.
  • Admin can set a price per day for every advertisement position.
  • Sellers and customers can purchase the advertisement display positions according to the availability.
  • A date range can be selected for which advertisement will be displayed and make payment accordingly.
  • Customers can only add Banners on the booked advertisement position.
  • Sellers can add Products and banners as per their choice on the booked advertisement position.
  • Sellers can add only their own products of the marketplace in the product carousel.
  • Admin can add any product in the product carousel.
  • Admin can view all the ad orders at one place.
  • Admin can select the maximum number of products to be shown on the product carousel according to ad block position.
  • Admin can configure whether to allow/disallow product carousel for each position from the backend.
  • Admin can configure the auto-approval settings for the products and banner added in the block content.
  • Admin can publish/unpublish the advertisements displayed on the website.
  • Admin can view the list of customers and sellers who have purchased the advertisement blocks.
  • Admin can approve/deny or delete the advertisement products added by the sellers.
  • A mail will be sent to the customer and seller on the approval of block content.

Odoo Marketplace Advertisement Manager

Website View of Odoo Marketplace Advertisement Manager

Advertisement block booked by customers are displayed at different positions on the website.

Sellers can check whether the product carousel for a particular position is allowed by admin or not.

Admin can add any product of shop from here.

Users can only add the banner in the ad block booked.

Seller View of Odoo Marketplace Advertisement Manager

Sellers have the option to add banner or products for the booked ad block position.

Sellers can only add their own approved products of the marketplace from here.

Admin View of Odoo Marketplace Advertisement Manager

Here, Admin can view all the ad blocks of the website.

Admin can view all the details of ad blocks lines along with content status.
