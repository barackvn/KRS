ODOO TODO List

Now, manage your tasks and To-DO list directly in the Odoo ERP system with Odoo ToDO List Module to increase proficiency. Never miss out any task, prioritize the assignment and maximize the efficiency.
The module adds a ‘TODO LIST’ menu to the Project App for the Odoo user to manage various upcoming tasks and assignment. A ToDo item for any running or upcoming project can be created and saved as a reminder without opening any task for the same. The ToDo item remains in a pending state until a task is created. Once a task is created from a ToDo item, its status changes to 'Done'.

Basic Features
  • It allows creating a ToDo list in the Odoo to serve as a reminder for any task.
  • A ToDo list can be created by the Odoo user for any project. The items in the list won’t show up in the tasks.
  • A ToDo list item gets done once a task is created for the same in the Odoo.
