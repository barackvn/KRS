Odoo Marketplace Seller Blogs

Odoo Marketplace Seller Blogs is a module by which sellers of the marketplace can post their blogs. A seller can create and edit their own blog posts.

Highlighted Features

  • The Admin and Seller can create blogs and post it on the website.
  • Admin can manage all blogs.
  • A seller can create and edit their blog posts.
  • Sellers will have a separate tab in their profile for their blogs.
  • Admin can set the configuration setting for auto approval of blog posts by the seller.
  • Admin can also give the permission to each seller separately for auto approval of blog posts.
  • Admin and seller can publish/unpublish the blog after its approval.
  • Facility to add tags based on different categories.
  • Customers can see seller's blog by visiting seller's profile page also.
  • Any user can comment on the blog post by the seller.

Odoo Marketplace Seller Blogs

Website View of Odoo Marketplace Seller Blogs

Admin can view the blog published on the website in the blog menu.

Admin can read the complete blog and its comments.

A seller can view the published blog under Blog menu on the website.

A new seller blog page will be visible on the seller's profile page showing

all the blogs of that particular seller.

Seller can view the complete blog here and view the comments below.

Admin can easily drag and drop the five latest blogs of marketplace Sellers on the website.

Admin can customize the desired blog view.

Five latest blogs of marketplace Sellers will be visible on the website.

Seller Side View of Odoo Marketplace Seller Blogs

A seller can view his blogs at one place.

Seller blogs record will be maintained under Blog Post menu.

Seller will enter the title, subtitle, tags, blog, and content of the blog.

A seller can request to approve his blog.

A seller can publish the blog on the website after its approval.

Admin Side View of Odoo Marketplace Seller Blogs

Admin can see all the details of blogs of seller at one place.

Admin can enable/disable auto publish a blog for a particular seller.

Blog Post menu gets visible to admin in seller dashboard.

Admin side view of all created blogs of marketplace sellers.

Admin can enter the details of the blog here along with the content.

Admin can publish the blog on the website after its approval.

Configuration View of Odoo Marketplace Seller Blogs

Admin can set the default settings for the blogs of sellers.
