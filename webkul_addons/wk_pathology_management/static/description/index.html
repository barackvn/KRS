Pathology Lab Management System

Pathology Lab Management System is a module that can be used in a pathology to book a test for the customer, enter customer's details and after conducting the test , give test reports to the customer. This module provides efficient management for pathology lab records. The concepts and design behind the Pathology Lab Management System module is to ensure that it meets the demanding requirements set out for an ideal software to manage today's busy pathology labs.

Highlighted Features

  • Pathology Lab Management System module can be used in any Pathology Center to Book a Test for customer.
  • Patient will be provided with the date at which the test will be conducted.
  • For any kind of check ups, a customer can visit pathology and get his checkup done.
  • Admin can approve or reject or set to pending the booked test.
  • Invoice can also be created according to the tests booked by customers.
  • Lab Technicians can perform the test for the customer.
  • On approving the test request, the records for all corresponding labtests for Diagnosis will be created.
  • Technician can be assigned in the diagnosis records who will perform the test.
  • Obtained Value can be entered post the completion of Diagnosis and set the diagnosis done.
  • Other useful details for Diagnosis Report can also be entered.
  • Facility of Mail Notification on Approval of Test Request and Reminder Mail Notification before the test request date.
  • Facility to download a report of test request and the Report containing the results of Labtests.
  • Records for Test Parameters, Sampletypes, Units, Labtest, Test Packages,etc can be created and stored.
  • Records for Patients, Pathologist, Technicians can also be created and stored.
  • Records for Laboratory centers and its Collection centers can be created and stored.

Requests & Diagnosis

A new Test Request can be booked directly from here.

All the details can be entered and labtests can be added into it.

On approving the request, the records for diagnosis will be automatically created.

After approving, the invoice for this patient can be created.

All the Test Request records can be seen from here.

All the Diagnosis records can be seen from here.

Assign any technician who will conduct this diagnosis.

After assigning the technician, set the diagnosis to processing.

Before setting the diagnosis to done, enter all the necessary details which are useful for report.

All the diagnosis records which are in done state can be seen here.

When all the corresponding diagnosis are in done state, then the state of this request will be automatically done.

Also, the diagnosis report will the test results can be downloaded or send to patient by email.

View of test request and Test diagnosis report downloaded after the state is done.

Persons & Labtests

Records of Patients with all the general information can be created and stored from here.

Records for Pathologist of Pathology can be created.

Records for Technicians of Pathology can be created.

All for Labtests that are conducted in pathology can be seen here.

Labtest can be created and all the details can be entered

All the Test Packages can be seen and created from here and labtests can be added in these packages.

Configuration

Records for Source from where the test request is made can be created here.

Records for Laboratory center can be created here.

Collection Centers of this Laboratory center can be added.

Create records for Collection Center and give a primary Lab center to it.

Records for Units of LabTest Parameters can be created here.

Records for Labtest Parameters can be created here.

Records of Information for patient before the test can be created here and add these in Labtests.

Records for different sample types for labTest can be created here.

Default configuration settings like Centers and Mail Notification can be edited and stored from here.
