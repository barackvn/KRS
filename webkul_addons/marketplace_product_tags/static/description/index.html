Odoo Marketplace Product Tags

Tag Keywords to Products for better Internal Search!

Internal Search Improvements with Product Tags

Once you get a visitor on your website then the next important thing is the ease of product search. Adding product tags is a great idea for better internal search.
Do you own a marketplace where multiple vendors can sell their products?
Now, you can create multiple tags and assign them to the products.

Custom Tags to Filter Out The Products

Once you get Create Tag and assign products to it so that customers can easily search the products with its tag on the website.
Moreover, you can also set its font style which includes font-family, font-weight and font-size. Publish the tags to show them on Odoo website.

Assign Multiple Tags to a Product

Select multiple tags for a product so that your customers can easily search that product by selecting those tags. Moreover, you can also set the maximum number of tags allowed in a website.
