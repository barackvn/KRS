Kanak Calendar Appointment

Supported Editions
Community
Enterprise

Kanak Calendar Appointment module is used to book appointment offered by a company on its e-commerce platform online in odoo.

Key Features

You can Choose Team Members.

Create Personalized Scheduling

Browse Appointment Calendar

View Appointment

You can set availability of days and timing of appointment to be booked.

User can reschedule or cancel the appointment.

When User take the appointment, an email will be sent both to the booker and appointee.

Exception create if appointee want to Block the appointment for a particular timing or for a whole day.

User can Book appointment in time slot, they can also book in multiple time slot.

All appointments are shown in my appoinment menu.

Enable Team Member Available for Appointment Checkbox

  • If you enable this checkbox, team members are shown on website.
Appointment Configuration

  • Set Availability of days and timing of appointment to be booked.
  • You can set Time slot of appointment.
  • Add lunch timing during appointment schedule.
  • Set max / min date for the calender to be visible.
Open Website

  • Appointment menu shown on Website.
  • Select team Member for taking appointment.
Select Date and Timing of Appointment

  • If any user select one of the date of appointment, another user unable to take appoinment on that time.
Fill Appointment Form

  • Add all details and click on confirm booking.
Appointment Confirmation Message Shown

Mail Sent

  • Once user Booked the appointment, email will sent both to the booker and appointee.
  • You have two options here-Reschedule or cancel the appointment.
  • If booker reschedule the appointment, time slot calender shown on website for take appointment again.
  • If booker cancel the appointment, popup shown to write the reason for cancelling the appointment.
  • You can see appointment details in my appointment tab.
Appointment Details

  • Admin can see the booking details of a user.
Booking Reschedule

  • Once user reschedule their booking, date time calender again shown on website to select one of the date of appointment.
Reschedule Confirm

Create Exception

  • If appointee want to block the schedule for a particular time or for whole day, he/she create a exception.
Exception Detail shown in my Exception

  • Admin can see exception details in order line.
Open Appointment Calender on Website

  • If appointee create exception for whole day,Here you can see user unable to take appointment on this day.
Appointment Details shown in my Appointment menu
