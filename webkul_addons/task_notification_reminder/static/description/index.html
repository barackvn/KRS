Task Notification Reminder

Managing with tight scheduling and multiple tasks can sometimes lead to losing focus towards priority tasks. Do not procrastinate and stay ahead of the schedule with Odoo Task Notifications Reminder Module.

It lets you prioritize your task by setting daily reminders for important tasks so you can manage them accordingly. The reminders can be chosen to be shown in the Odoo backend as well as through mail notifications. The admin can select the days before the task deadline to start receiving the daily reminders. Moreover, the reminders can be set for the entire project or a certain task in a project.

Basic Features
  • Create Notification Reminder for task deadline.
  • Notification Reminder creates for the project. Then, it covers all the tasks of that project.
  • Notification will be sent in which you are the Project Manager, Task Followers, Assigned User, Team Members.
  • Enable/disable email notifications.
  • Manage the number of days (before/after) the deadline of a task to start receiving daily notification reminders.
