Odoo Product Tags
Providing an easy interface for managing the tags for products and offering products filtration on the basis of tags

Salient Feature:

Create tags give them name and style.

Select products that will belong to each tag.

Manage limit on the number of tags for displaying on the website.

Describe your products using tag keywords.

Link similar products using tag keywords.

Product Tags Configuration and Setup

Put the tags limit here, which will display on the website.

Here you can manage the products tags.

Here, you can assign multiple products to a tag
and also set tag style and publish options.

You can also allocate tags on product model view.

Website View

A Customer can filter the products associated with a tag.

Admin can also unpublish the tags based product filter options.
