Odoo Reminder Mail

Stay Notified with Odoo Reminder Mail

Keep your staff and Odoo users updated with all the important dates and events with Odoo Reminder Mail Module.

Send memorandums and Daily Task reminder mails directly on the mail address of users with Odoo reminders.

Inform the users about deadlines, urgent tasks and more with simple automated reminder memos.

The reminder emails are sent directly to the user's registered email address in Odoo.

Features:

  • The module allows both Odoo Admin and users to set reminder emails in Odoo for internal users.
  • An Odoo reminder can be set for a single user or multiple users at once.
  • The reminders are sent to registered email addresses of the respective recipients from Odoo.
  • The admin can also choose to send multiple alerts for the same Odoo reminder.
  • The frequency of the multiple alerts can be set to daily, weekly and monthly basis.
  • The admin can also choose the start and end date of reminder alerts.
  • The history of the sent reminder emails can be viewed in the Odoo reminder form.
