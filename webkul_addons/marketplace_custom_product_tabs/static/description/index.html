Odoo Marketplace Custom Product Tabs

Provides facilty to Show More Product Details on Website Using Product Tabs.

Features

  • Admin can create Tabs for product of any seller on marketplace.
  • Sellers can create tabs for their own products of marketplace.
  • A Seller can describe Product using text, videos, images or tables.
  • View of Tabs on Product page in website can be in Vertical and Horizontal Type.
  • Allows to set the sequencing of the product tabs.
